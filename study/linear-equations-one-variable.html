Linear Equations in One Variable (SAT)

Get in, get the points: forms → translation → clean algebra → instant solution checks.

1) Know the forms

Slope–Intercept

y = m x + b shows slope m and intercept b immediately.

  • Use when: you have a graph, or are given slope & intercept directly.
  • Fast to compare slopes/intercepts to classify # of solutions.
  • Great for quick predictions/interpolations from a line.

Point–Slope

y - y₁ = m(x - x₁) builds a line from one point and a slope with zero fluff.

  • Use when: given a single point and a slope.
  • Skip finding the intercept unless needed; convert to slope–intercept only if necessary.

Standard

A x + B y = C keeps integers tidy and is great for combining equations cleanly.

  • Use when: you want elimination-friendly integers (avoid fractions).
  • Handy for quick coefficient comparisons and structured manipulation.

2) Translate words → equation (Blueprint)

Assign variables

  • Short, meaningful: p (pay rate), h (hours), x (unknown).

Keyword → operation

  • “plus”, “increased by”, “more than” → +
  • “minus”, “decreased by”, “less than” → −
  • “times”, “of” (as in % of) → ×
  • “per”, “each” → rate/division or multiplication by a unit rate
  • “is”, “are”, “equals” → =

Units & rates

  • Match both sides: dollars with dollars, miles with miles, etc.
  • Convert rates properly: ($/gal) ÷ (mi/gal) = $/mi

Build step-by-step

  • Write math for each sentence chunk, then simplify.
  • Confirm the target before solving: do they want x, x − 5, time, or base rate?

3) Execute algebra cleanly (advanced tips)

  • Clear fractions early: multiply both sides by LCM to simplify.
  • Group first, isolate second: combine like terms before moving across the equals sign.
  • Target expressions directly: if they want x − 5, isolate that expression (don’t solve for x unless needed).
  • Control negatives: rewrite subtraction as “add a negative” to reduce sign errors.
  • Mirror operations: whatever you do to one side, do immediately to the other.
  • Scan for structure: factor common terms once to save multiple steps later.
  • Quick system check: if two linear equations appear, compare slopes/intercepts first; don’t solve blindly.
